Simple Facebook Plugin which allows you to add Facebook Widgets to your Site using the Facebook SDK Api

== Description ==

Its a very simple Facebook Plugin which adds Facebook Template functionality.

== Installation ==

Upload the Simple facebook plugin to your blog and activate it. It would work as is.

== FAQ ==

= Is it compatible with latest WordPress? =

Yes, it is, as well as with the latest PHP.
